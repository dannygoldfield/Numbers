# Numbers Documentation Index

This directory contains design, system, and interface documentation for Numbers.

## Canonical Documents

These define system behavior and must not drift.  
Changes here alter the meaning of Numbers.

- `CORE-SEQUENCE.md`  
  Defines the invariant auction and inscription sequence.

- `PRD.md`  
  Defines non-negotiable system constraints.

- `ARCHITECTURE.md`  
  Describes system structure and component responsibilities.

## Interface Documentation

These describe how the system is presented.  
They must conform to the canonical documents but may evolve.

- `WEBSITE-PRD.md`  
  Product requirements for the Numbers website UI/UX.

- `VIEWER-DETERMINED-RENDERING.md`  
  Rendering constraints and presentation neutrality.
